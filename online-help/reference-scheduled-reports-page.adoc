= 「報告排程」頁面
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
「報告排程」頁面可讓您檢視已建立報告的詳細資訊、以及產生報告的排程。您可以搜尋特定報告、修改報告排程的特定屬性、以及刪除報告排程。

「報告排程」頁面會顯示已在系統上建立的報告清單。

* *排程名稱*
+
排程報告的名稱。此名稱一開始會包含檢視名稱和頻率。您可以變更此名稱、以更好地反映報告內容。

* *檢視*
+
用來建立報告的檢視。

* *收件者*
+
接收產生報告的使用者電子郵件地址。每個電子郵件地址都必須以英文分隔。

* *頻率*
+
產生報告並傳送給收件者的頻率。

* *格式*
+
報告是以PDF檔案或XLSX或CSV格式產生。

* *行動按鈕*
+
可供編輯或刪除報告排程的選項。
